# SemVer-Release

This is a simple semantic versioning tool written in Rust that allows semantic versioning to be setup in a project for automatic version updates on release. This project aims to be a straightforward and easy tool to use that doesn't require you to install a whole bunch of dependencies along with it.

A release binary will be made available in the Github repository for download that you can just run as a command line tool in a github action to perform semantic releases.
